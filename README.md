# erpTest
闲暇时间写的一些练手的小Demo

#### 启动
`java -jar`

#### 默认端口
`8081`
#### emoji转换地址
`127.0.0.1:8081/index`

### 功能:
* emoji转换
* minio上传下载测试
* Notion块添加内容测试
* 文字高亮
* 微信生成二维码
* 企业微信机器人推送
* ~~随机生成装备~~
* 新增功能 chatGPT
